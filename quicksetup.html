    Windows Environment Setup

    This guide shows how to get the auth up and running quickly in a windows 7 focused environment.

    Python Setup

    AllianceAuth is written in django which is a web framework around python. In order to begin development you must first install python and a few other things. Below are the links. Please pick the appropriate x86 or 64bit depending on your computer.

    1. Python 2.7.8
    2. Mysql-python
    3. Python-Mysql-Connector

    After you have those installed we need to add python to your system path.

  • Click the start icon and right click control panel
  • Click "Advanced System Properties"
  • Click "Environment Variables"
  • Scroll till you see the word path, click it and then click edit
  • At the very end of the string (moving to the right) add the following 
        ";C:\Python27;C:\Python27\Scripts" without the quotes
  • Click "OK" on all open windows

    • Now we have python installed we need to install pip to grab our other dependencies 

  • Download the get-pip.py script from here Get-Pip.py save it to your desktop
  • Open a command prompt 
        (Click start button then in the run bar type "cmd" without the quotes)
  • Naviagate to your desktop by typing "cd Desktop"
  • Run the get pip install by typing "python get-pip.py"

    • Grabbing a copy of the AllianceAuth

    Now that we have python setup we are going to grab the latest development branch from git, Open a command prompt and type the following 

  • git clone https://github.com/R4stl1n/allianceauth.git
  • cd allianceauth

    • AllianceAuth - Dependencies

    Now we can start working on getting the dependencies for the auth. This is easy first we have to make a quick change to requirements.txt 

  • Open requirements.txt
  • comment(add a "#" infront of it) or delete the line that says "mysql-python"

    • Open up a command prompt and type the following to get the dependencies 

  • pip install -r requirements.txt

    • AllianceAuth requires a mysql database to store its information in so we need to set one up. On windows there are a few options however the easiest is to use xampp 

  • Download xampp and install it -  Xampp Download (either one works)

    • AllianceAuth - Database Setup

    With xampp installed we now have a way to store our information. So lets create a mysql database. Note: if you cannot start the apache service due to a "port error" either stop the application using the port (ex. skype) or change the port in the apache config

  • Open the xampp application
  • Click the start button on both the apache and mysql service's
  • Click admin button on the mysql service line, this will open up your 
      web browser and take you to the phpmyadmin page. 
  • Click the button called databases at the top of the webpage
  • In the new database field enter the name "alliance_auth" then click 
      create new database
  • After the database is create it should appear in the left hand side navigation.
      Double click it
  • Click the privileges button at the top of the screen
  • Click the Add User Button 
  • In the username field and password field put the following "allianceauth",
      in the host field erase the percent sign and put in "localhost"
  • Scroll down the page and hit the go button

    • AllianceAuth - Configuration

    By default if you setup the database as instructed you won't need to modify the database connection information. Open an command prompt in the root file structure

  • copy alliance_auth/settings.py.example alliance_auth/settings.py

    • Now edit the settings.py file

  • Configure the ALLIANCE_ID settings in the settings.py 
      in the alliance_auth folder
  • Configure the CORPORATION_ID and CORPORATION_VCODE settings in the settings.py 
      in the alliance_auth folder

    • Now all we need to do is populate the database. Open the command prompt and navigate to the root folder of the project and enter the following commands. Note: make sure to create the super user.

  • python manage.py syncdb
  • python manage.py shell < run_alliance_corp_update.py

    • AllianceAuth - Startup

    Last we go ahead and startup the AllianceAuth. Open up the command prompt navigate to the root folder of the project.

  • python manage.py runserver

    • That is all it takes if for whatever reason you need to run the automated task inside of celerytask/tasks.py open a command prompt and navigate to the root directory 

  • python manage.py shell
  • from celerytask.tasks import taskfunctionhere
  • taskfunctionhere()
